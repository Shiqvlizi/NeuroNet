# NeuroNet

这是一个神经网络抽象.

## 它能做什么

- 这个项目更像是一本书, 目的在于为了给学习神经网络原理的人一些启发, 里面有丰富的注释 ~~(还没写)~~ 以及解耦的函数, 以至于可以学习原理
- 运行速度肯定是不如 `python` 的库的, 因为这个项目是 CPU 计算而非 GPU

## 你能做什么

- 你可以修改 `layerWidths` 来更改层数和每一层的宽度
- 你可以调用 `backPropagate` 来更新参数, 每次调用都会更新一次参数
  - 参数: `训练输入`, `标准输出`
- 你可以修改 `learnRate` 来决定神经网络每次更新参数
- 你可以传入自己的训练集来进行训练
- 阅读代码来了解原理


## 为什么使用 `C++`
- 想手动实现所有细节, 从矩阵乘法到反向传播
